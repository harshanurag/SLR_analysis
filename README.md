## Sea Level Rise Projection Analysis

This project analyzes sea level rise projections from the IPCC AR6 report. The analysis is done for the closes lat, long coordinates for Dubai.

Data Source: https://sealevel.nasa.gov/ipcc-ar6-sea-level-projection-tool?lat=25&lon=%2054&data_layer=scenario


### Output

The analysis generates the following plots:
- `output/all_scenarios_median.png`: Median sea level rise projections for all scenarios
- `output/ssp126_with_uncertainty.png`: SSP1-2.6 scenario with uncertainty bounds
- `output/ssp245_with_uncertainty.png`: SSP2-4.5 scenario with uncertainty bounds
- `output/ssp370_with_uncertainty.png`: SSP3-7.0 scenario with uncertainty bounds
- `output/ssp585_with_uncertainty.png`: SSP5-8.5 scenario with uncertainty bounds
